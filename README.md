🏴‍☠️ BountyBoard — Hunt or Be Hunted
Turn your server into a wild west with fully craftable, player-driven bounties. Place a price on anyone's head — in any items you choose.

✨ Features

🪵 Craftable Bounty Board — craft it, place it, done
🖥️ Intuitive GUI — type a target name and drop reward items directly into the board
💀 Kill to Collect — slay the wanted player and the rewards go straight to your inventory
📢 Server Broadcasts — everyone knows when a bounty drops
🔴 Wanted Effect — wanted players are visually marked for all to see
💾 Persistent Data — bounties survive server restarts
🛡️ Admin Commands — /bountyboard give | list | clear <player>

📖 How to Use

Craft the Bounty Board and place it anywhere in the world
Right-click it to open the bounty GUI
Click "Set Target" and type the name of the player you want to put a bounty on in chat
Place any reward items you want into the reward slots (emeralds, diamonds, weapons — anything goes)
Hit "Post Bounty" — the items are locked in and the bounty goes live
The wanted player will be marked with a visible effect so hunters can track them down
Kill the wanted player → rewards are automatically sent to the killer's inventory
A server-wide message announces the kill and who claimed the bounty
